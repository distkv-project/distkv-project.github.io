Distkv

A light weight distributed key-value database system with table concept.

Quick StartView on Github

abstract

Distkv project is a distributed key-value database system. Besides these features, Distkv supports table concept which looks like tables in relational databases.

Feature List

  • easy use

    Redis-like data structure

  • table

    Table concept based on kv store

  • consistency

    Strong consistency model

  • client

    Easy to use clients

  • web

    Strong ecosystem for Web application and micro-service
